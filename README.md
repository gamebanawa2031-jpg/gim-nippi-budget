# Gim & Nippi Budget

A premium, shared financial management suite built with React, Firebase, and PWA features.

## Features
- **Dashboard**: Real-time overview of income, expenses, and balances.
- **Wedding Planner**: Independent budget tracking and task management for the big day.
- **Loan Manager**: EMI calculations, amortization schedules, and overpayment insights.
- **Goal Tracker**: Savings goals with progress visualization.
- **Insights**: Advanced spending analysis and trends.
- **Shared Access**: Real-time sync across devices using Firebase Firestore.
- **PWA Support**: Installable on iOS and Android home screens.

## Deployment
This project is configured for easy deployment on **Vercel**.

1. Connect this repo to Vercel.
2. Add your Firebase environment variables if needed (though they are currently in the code for simplicity).
3. Deploy!
